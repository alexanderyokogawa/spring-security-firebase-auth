### Dependencies
`<dependency>`

`<groupId>com.google.firebase</groupId>`

`<artifactId>firebase-admin</artifactId>`

`<version>9.1.1</version>`

`</dependency>`

Download the service account key from firebase console and place it in the resources' folder.
